Install
------------------------------------------------------------------------------

``claude_desktop_config`` is released on PyPI, so all you need is to:

.. code-block:: console

    $ pip install claude-desktop-config

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade claude-desktop-config
